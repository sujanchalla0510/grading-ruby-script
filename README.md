## Orca-Grading
```
- This script will automatically download all the students assignments from the url https://www.cs.usm.edu/moodle/
- It Creates a separate folder for each of them in the local system and stores their homeworks in their respective 
   folders.
- It automatically compiles,run the homework files and if there are any errors in the homework
   it will automatically send the feedback to the student mail id.
```
