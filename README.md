## Orca-Grading
```
1. This script will automatically download all the students assignments from the https://www.cs.usm.edu/moodle/
2. It Creates a separate folder for each of them in the local system and stores their homeworks in their respective 
   folders.
3. It automatically compiles,run the homework files and if there are any errors in the homework
   it will automatically send the feedback to the student mail id.
```
